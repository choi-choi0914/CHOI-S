# CHOI'S Edit: Jeonju Cafe Curation

바닐라 자바스크립트(Vanilla JS)로 구현한 전주 카페 큐레이션 & 추천 서비스입니다.

## 🛠 주요 기능
* **카테고리 필터링:** 분위기, 커피, 디저트별 모아보기
* **실시간 검색:** 카페 이름으로 즉시 검색 기능
* **찜하기 (Local Storage):** 마음에 드는 카페 저장 및 모아보기 (브라우저 종료 후에도 유지)
* **랜덤 추천:** 결정 장애를 위한 랜덤 카페 추첨 게임
* **반응형 디자인:** 모바일과 데스크탑 모두 지원
